# Agile Playbook

A practical playbook for Agile transformation, delivery excellence, coaching, culture, and metrics.

## Contents
- docs/playbook: Core playbook chapters
- docs/transformation: Transformation approach and governance
- docs/metrics: Metrics definitions, dashboards, and interpretation guidance
- docs/culture: Cultural barriers, change management, and team behaviors
- docs/coaching: Coaching structures, engagement models, and checklists
- docs/raci: RACI models for common Agile operating models
- docs/diagrams: Swimlane diagrams and other visuals (stored as Mermaid where possible)
- docs/templates: Playbook templates (retros, working agreements, etc.)
- docs/artifacts: Downloadable artifacts (example charters, canvases, etc.)

## Conventions
- Markdown only, ASCII only.
- Prefer Mermaid for diagrams where possible.
- Keep templates copy/paste friendly.
