# Style Guide (ASCII only)

Rules:
- ASCII characters only. Avoid smart quotes, long dashes, and special bullets.
- Use "-" for bullets and "--" only in code blocks.
- Use "->" instead of arrows.
- Use simple headings: "#", "##", "###".
- Keep lines under 120 chars when practical.
- Diagrams: prefer Mermaid in markdown code fences.

  A[Start] --> B[Do work] --> C[Finish]
